Современный корпоративный сайт ОАО *«Барановичский завод автоматических линий»*, разработанный на Laravel.
Проект представляет собой информационный веб-сайт предприятия с системой управления контентом, позволяющей публиковать новости, управлять каталогом продукции и редактировать основные страницы сайта.

# Основные возможности
- Главная страница предприятия
- Каталог продукции
- Категории продукции
- Страница товара
- Новости предприятия
- История предприятия
- Наша команда
- Контакты
- SEO-оптимизация страниц
- Адаптивная верстка
- Административная панель
- Загрузка изображений
- Управление контентом

# Технологии
Backend: Laravel, PHP, Eloquent ORM, Artisan, Composer
Frontend: Blade, Bootstrap, HTML, CSS3, JavaScript
Database: MySQL

Администратор может:
